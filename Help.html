The use of tables

You may have noticed that the tables for the different inventory items are quite similar. They are built in the same way with some different features and variations in styling. This makes the tables easy to understand for its purpose.

Storage location

Each table has comment input fields that allow users to add notes or details about the specific inventory item. This feature is particularly useful for keeping track of the storage location of each item on the vessel.

Expiry dates

Some tables like provisions, medicine-chest and safety equipment have expiry dates associated with their items. Simply enter the expiry date in the designated field, and the system will automatically track and notify you of upcoming expirations with colors in the input fields.

Notice❗ The lists are not ment as guidance on routine checks or function tests. Please refer to platforms approved by your maritime authority and radio provider. You can find links to platforms that offers safety management in the link box.

Set minimum and maximum stock levels

Avoid stockouts and overstock situations by setting minimum and maximum stock levels for each item. This ensures you have the right amount of inventory on hand at all times.

Upload images

For better inventory management, you can upload images of each item. This visual reference can help with identification and tracking.

Order new

Place a new order, send, print or take the order from the entire list to orders for checking the unit price and total amount. Prices must be typed manually.

Flexiblebility

The system is designed to be flexible and you don't have to type in every input field for the system to work and save your progress.
